# Code Arena — Inferno '26 🏆

**Code Arena** is the premier coding competition hosted during **Inferno '26**, the annual flagship festival of **SDJ International College**. This platform is the official gateway for participants to showcase their logic, syntax, and problem-solving speed.

Designed and engineered by the **Semicolon IT Wing**, the website handles real-time competition management, participant tracking, and challenge delivery.

---

## 🏛️ Event Overview
* **Fest:** Inferno 2026
* **Competition:** Code Arena
* **Host:** SDJ International College
* **Managed By:** Semicolon IT Wing

## 🚀 Key Features
* **Real-time Leaderboard:** Dynamic ranking system to track top coders during the event.
* **Challenge Portal:** A curated set of problems ranging from beginner logic to advanced algorithms.
* **Modern UI/UX:** A high-performance interface built for speed and focus, utilizing a dark-themed aesthetic.
* **Responsive Design:** Optimized for all devices to ensure participants can access information on the go.

## 🛠️ Technology Stack
* **Core:** React.js & TypeScript
* **Design:** Tailwind CSS & shadcn/ui
* **Performance:** Vite-powered frontend for near-instant load times
* **Hosting:** Vercel (Live Production Environment)

## 📂 Project Structure
* `client/` — Core application logic and competition UI components.
* `shared/` — Shared data structures, validation logic, and type definitions.
* `public/` — Official Inferno '26 assets and SDJ College branding materials.
