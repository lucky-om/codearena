# Security & Fair Play Policy — Code Arena

## 🛡️ Competition Integrity
Code Arena is a strategic competitive environment. To ensure a fair experience for all teams during **Inferno '26**, the following security measures are active:

* **Authentication & Whitelisting:** Access is restricted to pre-registered Team IDs. Unauthorized entry attempts are logged and flagged.
* **State Synchronization:** Strategic Card actions (Pause, Point Exchange, Swipe) are verified against a backend state to prevent "double-spinning" or multi-use glitches.
* **Input Sanitization:** All user inputs within the terminal and quiz modules are filtered to prevent injection attacks.

## 🚫 Anti-Cheat Protocol
Any attempt to bypass the game logic, manipulate the Strategic Card cooldowns, or reverse-engineer the scoring API will result in immediate disqualification from the event.

If you discover a technical bug or a bypass that could impact the fairness of the competition, please report it immediately:
